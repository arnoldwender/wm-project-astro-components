
## Quick Start

```astro
---
import { SEO } from '@wendermedia/astro-components/seo';
import { Header, Footer } from '@wendermedia/astro-components/layout';
---

<html lang="en">
  <head>
    <SEO title="My Site" description="Built with @wendermedia/astro-components" />
  </head>
  <body>
    <Header siteName="My Site" navItems={[{ label: 'Home', href: '/' }]} />
    <main><slot /></main>
    <Footer siteName="My Site" />
  </body>
</html>
```

## Why This Library?

- **WCAG 2.1 AA** — Full accessibility with BFSG 2025 compliance, keyboard navigation, ARIA labels, screen reader support
- **GDPR/DSGVO Ready** — Privacy-first design, consent gates on YouTube, Google Maps, analytics, zero tracking by default
- **TypeScript First** — Complete type definitions, full IntelliSense, `export interface Props` on every component
- **Design Token System** — CSS custom properties for colors, spacing, typography, shadows, dark mode built in
- **Astro 4.x & 5.x** — View Transitions, Content Collections, zero client JS, React/Vue/Svelte/Solid integrations
- **SEO Optimized** — Schema.org structured data, Open Graph, Twitter Cards, hreflang, canonical URLs, semantic HTML
