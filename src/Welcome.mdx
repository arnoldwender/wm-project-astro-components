# @wendermedia/astro-components

**158 production-ready, accessible, GDPR-compliant Astro components.**

Built by [Wender Media](https://www.wendermedia.com) — Web Agency from Halle (Saale), Germany.

---

## Getting Started

```bash
npm install @wendermedia/astro-components
```

```astro
---
import { SEO } from '@wendermedia/astro-components/seo';
import { Header, Footer } from '@wendermedia/astro-components/layout';
---

<html lang="en">
  <head>
    <SEO title="My Site" description="Built with @wendermedia/astro-components" />
  </head>
  <body>
    <Header siteName="My Site" navItems={[{ label: 'Home', href: '/' }]} />
    <main><slot /></main>
    <Footer siteName="My Site" />
  </body>
</html>
```

## Key Features

- **WCAG 2.1 AA** — Full accessibility with BFSG 2025 compliance
- **GDPR/DSGVO** — Privacy-first, consent-based third-party loading
- **TypeScript** — Complete type definitions for all components
- **Design Tokens** — CSS custom properties for theming and dark mode
- **Zero Tracking** — No third-party analytics by default
- **Astro 4.x & 5.x** — View Transitions and Content Collections ready
